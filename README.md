# DFIR CTF Challenge Development

This repository contains Digital Forensics challenges I designed for the ShadowBrokers CTF organized by GDGoC Future Academy.

The challenges simulate real-world DFIR scenarios, focusing on evidence discovery, analysis, and reconstruction using industry-relevant tools and techniques.

## 🔍 Challenges

### 🧟 Walking Dead – Disk Forensics
A raw disk image with manipulated partition tables where hidden ext4 and NTFS partitions contain fragmented evidence.

### 🖼 GDGCO – Steganography / Forensics
A JPEG image with embedded archive data containing Base64-encoded fragments of the flag.

---

## 🧠 Skills Demonstrated
- Disk & partition analysis (MBR, hidden partitions)
- File system forensics (FAT32, NTFS, EXT)
- File carving & data recovery
- Steganography detection & extraction
- Evidence reconstruction
- DFIR investigation workflow

---

## 🛠 Tools & Techniques
- mmls, fdisk
- binwalk, strings
- Autopsy, FTK Imager
- Base64 decoding
- File carving tools
