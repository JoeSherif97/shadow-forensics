# 🖼 GDGCO – Steganography Challenge

## Category
Steganography / Forensics

## Difficulty
Medium

## Description
A JPEG image that appears normal but contains hidden data. Upon deeper inspection, the file includes an embedded archive containing multiple files with encoded fragments of the flag.

Participants must extract hidden content and reconstruct the final flag.

## Skills Tested
- Detecting hidden data in files
- Extracting embedded archives
- Base64 decoding
- Multi-step evidence reconstruction

## Tools Suggested
binwalk, strings, CyberChef, base64
